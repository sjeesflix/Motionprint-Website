Frequently asked questions

What operating systems does Motionprint Ergo run on?

Motionprint Ergo is currently a Windows desktop application. A macOS version is in development. For a full overview of system requirements, see the download page.

How many seats are included in a licence?

One licence key covers 2 simultaneous device activations — for example a laptop and a desktop. Team licences are priced per user seat. Contact us if you need a multi-site arrangement or have questions about your specific setup.

Are new assessment methods included in my subscription?

All assessment methods and exports currently in Motionprint Ergo are included with your active licence. New features are added regularly — most are rolled out across all plans. As the platform grows, some advanced modules may be introduced as part of separate subscription tiers.

Do I need an Xsens suit to use Motionprint Ergo?

Yes. Motionprint Ergo is purpose-built for Xsens motion capture data. It processes .mvnx files exported from Xsens MVN and supports MVN Awinda and MVN Link suit configurations.

How does licence activation work?

Each licence key is bound to your machine's hardware fingerprint. If you need to move to a new machine, you can deactivate an existing activation from within the app. Licence validation requires an occasional internet connection — there is a 30-day offline grace period, after which the app switches to read-only mode until it reconnects.

Is Motionprint Ergo GDPR compliant?

Yes. All data — motion recordings, analysis results and reports — is processed and stored locally on your machine and never transmitted externally. The licence validation server is GDPR compliant.
