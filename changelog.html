Release notes

Changelog

A full history of every Motionprint Ergo release — what was added, improved and fixed in each version.

Versions
  • v1.0.0
v1.0.0 March 2026
Initial release

Data input

  • MVNX file import — Direct import of .mvnx files exported from Xsens MVN. Compatible with MVN Awinda and MVN Link suit configurations. No conversion or pre-processing required.
  • Task event definition — Define one or more task events within a recording to segment and assess individual lifting cycles or handling operations.

Assessment methods

  • NIOSH Lifting Equation — Calculates the Recommended Weight Limit (RWL) and Lifting Index (LI) for single-task lifting. Multi-task sessions produce a Composite Lifting Index (CLI) across all defined task events.
  • KIM-LHC — Key Indicator Method for Lifting, Holding and Carrying (BAuA standard). Evaluates physical overload risk based on load weight, duration, posture and working conditions, and outputs a risk score with action level classification.

In-depth analysis

  • 3D motion playback — A full-body avatar replays the captured motion data. Supports 1, 2, or 4 viewports in horizontal, vertical, or quad layouts, each with an independent camera angle and free orbit, pan, and zoom.
  • Body risk indicators — Color-coded markers on key joints update every frame to reflect current biomechanical risk: green (low), orange (medium), or red (high).
  • Score-per-frame graph — Time-series chart of the selected variable with a three-zone color-coded background. Click or drag to scrub to any frame; the avatar and all panels update in sync.
  • Variable selector — Switch the plotted variable to drill into individual ergonomic factors. NIOSH: LI / STLI, horizontal distance, vertical distance, asymmetry angle. KIM-LHC: total score, load handling conditions, trunk twist, load distance, hand height, hand/arm position.
  • Frame insights panel — Shows the numeric value of the selected variable at the current frame, a risk bar indicating where it falls across the three zones, and a zone-distribution pie chart showing the percentage of time spent in each risk zone across the full task.
  • Playback controls — Play/pause at 0.5×, 1×, and 2× speed, with frame range trimming for isolating specific parts of a recording.
  • Critical moments — Auto-detected (peak-score frame, sustained high-risk windows) and user-created (mark the current frame or select a range directly on the graph). Each moment is editable with start/end frames and a description, and appears as a shaded overlay on the graph.

Assessment configuration

  • NIOSH settings — Reference mass (LC) selectable from four standard options (EN 1005-2, NIOSH RNLE / ISO 11228-1:2021, 15 kg, 10 kg); analysis mode switchable between Peak Moment (worst-case across all frames) and Origin/Destination (traditional pickup and placement only); ISO 11228-1:2021 Annex E multipliers individually togglable for two-person lifting, additional physical tasks, one-hand lifting, and extended shift duration.
  • KIM-LHC settings — Fully editable load weight scoring table with separate thresholds for men and women; optional averaged overview across all tasks in the session for consistent load profiles.
  • Auto-generated critical moments — Toggleable for both methods. When enabled, automatically detects the peak-score frame and sustained high-risk windows in the recording.
  • Custom findings sections — Add, remove, rename, and reorder the sections in the report findings page, for both methods independently.
  • Filename templates — Configurable PDF and Excel filename patterns using variables such as {workstation}, {record_date}, and {operator}.
  • Privacy toggles — Optionally hide gender, operator name, or evaluator name from the interface, reports, and exports.

Reports & export

  • PDF export — Fully structured ergonomic report with custom branding — add your own logo and organisation name for client delivery and H&S documentation.
  • Excel export — Underlying assessment data exported to Excel for further analysis, archiving, or integration with your own reporting workflows.

Platform & installation

  • Windows 10 / 11 support — 64-bit installer for Windows 10 and Windows 11. Approximately 80 MB download. Per-user installation — no administrator rights required.
  • Licence activation — Activate with a Motionprint licence key (format: MP-XXXX-XXXX-XXXX-XXXX). Requires an internet connection for initial activation and periodic licence validation.
