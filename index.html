Supported methods

Built on validated
ergonomic standards

There are two validated methods available now, with more continuously being added to the platform.

Available now
NIOSH
For evaluating manual lifting and lowering tasks

Calculates the Lifting Index (LI) for single-task lifting or the Composite Lifting Index (CLI) for multi-task scenarios — quantifying the physical demand of manual lifting operations.

Learn more →
Available now
KIM-LHC
For assessing lifting, holding and carrying tasks

German BAuA standard for evaluating the risk of physical overload from manual handling tasks — based on load weight, duration, posture and working conditions.

Learn more →
