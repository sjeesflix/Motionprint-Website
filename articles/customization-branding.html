Platform Features

Customization & Branding

5 min read
Last updated February 2026
Platform Features
In this article

  • 2. Color Settings

Back to Knowledge Base

Motionprint Ergo lets you tailor the look and feel of your reports so they carry your organisation's identity when shared with clients or management. This article walks through every customization option — from logo and color settings to report layout and the settings that influence how results are displayed.

1 Branding Options

All branding and customization settings are found under Settings → Branding & Reports in the top menu bar. Changes apply immediately to all future exports and are saved per installation.

Screenshot

Company Logo

Upload your organisation's logo and it will appear in the header of every exported report — both PDF and Excel. Click Upload Logo and select your file.

  • Supported formats: PNG, JPG, SVG
  • Recommended size: 300 × 100 px, transparent background (PNG or SVG preferred)
  • Max file size: 2 MB

A live preview shows exactly how the logo will appear in the report header. To remove a logo, click the trash icon next to the current upload.

Screenshot

Two-part screenshot: left side shows the upload area with an uploaded logo thumbnail and the remove/replace controls; right side (or inset) shows the resulting report header with the logo in the top-left corner. A simple arrow or bracket connecting the two halves makes this clear.

Company Name

Enter your organisation's name in the Company Name field. This text is used in:

  • The report header, displayed below the logo
  • The PDF footer alongside the export date and page number
  • The "Prepared by" line on the optional report cover page

2 Color Settings

The color settings control the accent color used throughout your exported reports — chart fills, section dividers, data labels, and neutral UI elements inside the PDF.

Primary Accent Color

Click the color swatch next to Accent Color to open the color picker. You can enter a hex value directly (#EA523B) or use the picker to choose visually. The report preview on the right-hand side updates in real time.

Note

Risk-level colors (green, amber, red) are standardised by the NIOSH and KIM-LHC methodologies and cannot be changed. Your brand accent color applies only to neutral and informational elements.

Report Color Scheme

Choose the overall background style for PDF exports:

  • Light (default) — white background, dark text, colored accents. Best for print and most digital sharing scenarios.
  • Dark — dark-grey background, light text, colored accents. Works well for on-screen presentations.

3 Report Customization

These settings control which sections appear in exported reports and how the overall layout is structured. They apply to both PDF exports and the on-screen report viewer.

Cover Page

Toggle Include Cover Page to add a title page to every PDF export. When enabled, you can configure:

  • Report Title — default: "Ergonomic Assessment Report"; customizable per export or as a global default
  • Project / Client Reference — optional free-text field that appears below the title
  • Assessor Name — the name shown in the "Prepared by" line
Screenshot
Cover page configuration and preview
Split view: left side shows the cover page settings panel with the toggle enabled and all fields filled in; right side shows the resulting PDF cover page with logo, report title, client reference, date, and "Prepared by" line visible.

Included Sections

Use the Report Sections checklist to choose which content blocks are included in PDF exports. This lets you tailor reports for different audiences — for example, a summary-only version for management versus a full data export for an ergonomics specialist.

  • Score Summary (always included, cannot be disabled)
  • In-Depth Analysis tab
  • Findings & Recommendations
  • 3D Visualization screenshots
  • Raw data appendix

Header & Footer

Customize what appears in the running header and footer of each report page:

  • Header: logo (left), report title (center), export date (right) — each element can be toggled on or off individually
  • Footer: page number, company name, optional confidentiality statement (free-text, max 80 characters)

4 Settings That Influence Calculations

Motionprint Ergo's NIOSH and KIM-LHC calculations follow the published, standardised methodologies — the core algorithm is fixed. However, a handful of application-level settings affect how results are classified and displayed.

Important

None of these settings change the underlying ergonomic calculation. They only affect how thresholds are visualised and labeled in your reports.

NIOSH — Organisation Alert Threshold

The standard NIOSH guidelines define a Lifting Index (LI) of 1.0 as the point at which risk becomes present. In Settings → Assessment → NIOSH, you can optionally set a custom Organisation Alert level — for example, LI ≥ 0.85 — which triggers an early advisory badge in the report before the formal LI 1.0 threshold is reached.

This is useful if your organisation's H&S policy requires intervention before the standard threshold. The LI value itself is unchanged; only the advisory color and label in the report are affected.

Screenshot
NIOSH threshold settings panel
Screenshot of Settings → Assessment → NIOSH showing the default threshold markers (LI 1.0 and LI 3.0) and the optional "Organisation Alert" input field set to a custom value (e.g., 0.85). A small inset or annotation should show how the custom threshold appears as a yellow advisory band in the Score Summary chart.

KIM-LHC — Risk Score Display

KIM-LHC risk scores range from 1 to 4 with fixed action-level definitions. You cannot adjust the thresholds, but in Settings → Assessment → KIM you can configure whether the report shows:

  • The numeric risk score only
  • The action-level text description only
  • Both (default)

Units & Locale

Found under Settings → General:

  • Weight unit: kilograms (kg, default) or pounds (lb). Applies to all weight inputs and report outputs.
  • Date format: DD/MM/YYYY or MM/DD/YYYY
  • Language: English (default). Additional report languages may be added in future releases.
