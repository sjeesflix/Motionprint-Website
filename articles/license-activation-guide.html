License & Activation Guide

This guide explains how to activate your Motionprint Ergo license, manage seat assignments across devices, and transfer your license to a new machine. The licensing system is designed to be secure and flexible — supporting both online and offline activation, multi-seat plans, and straightforward seat reassignment.

1 License Activation

Your license key is provided by email upon purchase. It follows the format below and is entered directly in the application during first launch.

MP-XXXX-XXXX-XXXX-XXXX

Online activation

Online activation is the standard method and completes in seconds. The application communicates with the Motionprint License Server to validate your key and register your device.

Open Motionprint Ergo. The activation screen appears automatically on first launch.

Enter your license key in the format MP-XXXX-XXXX-XXXX-XXXX and click Activate.

The app submits your license key and machine fingerprint to the License Server for validation.

On successful validation, the license is activated and the application opens.

Offline activation

If the machine running Motionprint Ergo does not have internet access, you can activate offline using an activation file provided by Motionprint support.

In the activation screen, select Offline Activation and copy the machine fingerprint shown.

Contact Motionprint support with your license key and machine fingerprint.

Receive the activation file (.mpact) from support and save it to the machine.

In the app, click Upload Activation File, select the file, and the license will be activated.

Offline grace period

Once activated, Motionprint Ergo can operate without an internet connection for a limited grace period. The app checks in with the License Server periodically when connectivity is available to renew this period.

2 Seat Management

Each Motionprint Ergo license includes a fixed number of seats — the maximum number of devices that can be simultaneously activated under that license. Seats are assigned automatically when a device is activated.

Checking available seats

You can view the number of activated seats and remaining capacity at any time from within the application:

  • Open Motionprint Ergo and navigate to Settings → License
  • The License Management page shows your plan, total seats, and how many are currently in use
  • For API-based seat queries, use the GET /api/v1/license/status endpoint with your license key

Revoking or reassigning a seat

To free a seat — for example, to move a license from one machine to another — deactivate the license on the current device first, then activate it on the new one.

On the machine you want to free, go to Settings → License and click Deactivate License.

Confirm the deactivation. The seat is released immediately and becomes available.

Activate the license on the new machine using the same key via online or offline activation.

3 License Transfer

Transferring your license to a new machine follows the same deactivate-then-activate pattern. The license key itself does not change — it is reused on the new device.

Transfer steps

1
Deactivate on the old machine

In Motionprint Ergo on the old machine, go to Settings → License → Deactivate License. This releases the seat and unlinks the machine fingerprint from your license.

2
Activate on the new machine

Install Motionprint Ergo on the new machine, launch it, and enter the same license key. The activation completes online or offline as described in Section 1.

Deactivation via API

For automated or scripted workflows, deactivation can also be performed via the API endpoint:

POST /api/v1/deactivate — supply your license key and machine fingerprint in the request body. On success, the seat is freed immediately.

Seat limit

Transfers are limited by your plan's seat count. If all seats are in use and you cannot deactivate the old machine (e.g. it is no longer accessible), contact Motionprint support to manually release the seat.

4 Implementation

The licensing system in Motionprint Ergo is built on a server-side License Server that validates keys, tracks activations, and manages seat counts. Device identity is established using machine fingerprints so that licenses are securely bound to specific hardware configurations.

How validation works

When you activate online, the application sends your license key and machine fingerprint to the License Server over an encrypted connection. The server checks the key's validity, verifies the seat count has not been exceeded, and returns a signed activation token. This token is stored locally and presented on each application launch to verify the license without requiring a constant internet connection.

Access methods

Method Description Best for
GUI In-app license management under Settings → License Day-to-day activation, seat checks, deactivation
CLI generate_license.py command-line tool Scripted activation and advanced users
API REST endpoints for activation, status, and deactivation Integration with deployment pipelines or IT management systems

Key features

Secure activation — license keys are validated over an encrypted connection to the License Server; activation tokens are signed to prevent tampering.

Flexible seat management — seats can be reassigned at any time by deactivating and reactivating, with no need to contact support for routine transfers.

Offline support — once activated, the application continues to function during internet outages for a defined grace period without requiring constant connectivity.

API access — all licensing operations are available via REST API, enabling integration into automated deployment or IT management workflows.
