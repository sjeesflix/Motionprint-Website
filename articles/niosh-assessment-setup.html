Using Reports

Running a NIOSH Assessment

Before Motionprint Ergo can calculate a NIOSH assessment result, you must configure the input parameters. This article walks through the assessment setup page — every field, every step, and every control — so you can prepare a recording for analysis with confidence.

Overview

The NIOSH assessment input page is a single, scrollable screen with two main regions: a left panel containing a step-by-step form and a right panel displaying an interactive 3D viewport with playback and timeline controls.

The form is divided into four sequential steps that must be completed in order:

  1. Upload Recording — Load an MVNX motion capture file.
  2. General Information — Enter workplace, operator, evaluator, and other metadata.
  3. Additional Modifiers — Apply conditional multipliers (appears only if enabled).
  4. Trim & Define Lifting Tasks — Set the observation window and mark individual lifting actions.

Each step must be completed before moving to the next. The Run Assessment button at the bottom remains disabled until all required fields are filled and the assessment is ready to execute.

Real-time validation

Motionprint validates all entries as you type. If a required field is missing or invalid, a clear error message appears and the Run button stays disabled. This prevents incomplete assessments from being submitted.

1 Upload Recording

Motion capture data must be provided as an MVNX file — the Xsens motion capture format. This is the only supported input format for NIOSH assessments in Motionprint Ergo.

Click the file upload control to browse your system and select an MVNX file. Once a valid file is loaded, the 3D viewport on the right instantly renders the recorded movement and the timeline populates with the full duration of the recording.

No other steps in the form become accessible until a valid recording file is loaded. If you attempt to load an invalid or corrupted file, Motionprint displays an error message and prompts you to try again.

MVNX format requirement

If you are recording with Xsens MotionCapture, export the recorded session as an MVNX file before importing into Motionprint. The MVNX format preserves all motion data and skeletal information needed for biomechanical analysis.

2 General Information

This step collects metadata and contextual details about the assessment session. Some fields are always required; others appear or disappear based on your NIOSH report settings.

FieldRequired?Description
Workplace Yes Name or description of the workstation being assessed (e.g. "Assembly Line Station 3" or "Warehouse Packing Area"). Required text field.
Operator Conditional Name or ID of the worker being assessed. Hidden if "Hide operator name" is enabled in NIOSH report settings to protect privacy.
Evaluator Conditional Name or ID of the ergonomist or assessor conducting the evaluation. Hidden if "Hide evaluator name" is enabled in settings.
Gender Conditional Worker gender (radio buttons: Male / Female). Conditional based on "Gender not required" setting. Used to refine certain biomechanical calculations.
Working time Yes Total hours and minutes spent at the workstation per shift. Required field with separate inputs for hours and minutes. Used to determine the Frequency Multiplier duration category (see below).
Comments No Optional multi-line text field for notes, observations, or contextual information about the assessment.

Working time and duration category

The Working time value you enter directly determines which Frequency Multiplier (FM) lookup table is used during assessment. Motionprint automatically categorizes the shift duration as follows:

  • Short duration — ≤ 1 hour at the workstation.
  • Moderate duration — Greater than 1 hour, up to 2 hours.
  • Long duration — Greater than 2 hours (shifts of 2+ hours).

Each category has its own FM lookup table; the appropriate table is selected automatically once you confirm the working time.

3 Additional Modifiers

Additional Modifiers is a conditional step that only appears if modifiers are enabled in your NIOSH report settings. If your settings do not include any modifiers, this step is skipped entirely and you proceed directly to the trim and lifting tasks step.

When enabled, this step allows you to apply multiplicative adjustment factors that account for special lifting conditions not captured by the standard NIOSH variables. Each modifier applies a specific multiplier:

ModifierFactorApplies when
Two-person lift Pf = 0.85 The load is lifted by two workers simultaneously, sharing the weight.
Additional physical tasks Tf = 0.80 The worker performs other strenuous tasks (carrying, pushing, pulling) in addition to lifting.
One-hand lifting Of = 0.60 A lift is performed with only one hand instead of two.
Extended duration eM The work shift exceeds 8 hours. The multiplier value depends on shift length.

Select or check the modifiers that apply to the observed work. Multiple modifiers can be selected at once, and their effects are multiplicatively combined in the final RWL calculation.

4 Trim & Define Lifting Tasks

This is the most complex step and the one with the most interactive controls. You use the 3D viewport and timeline to define the precise observation window and to mark individual lifting tasks within that window.

The 3D viewport and playback controls

On the right side of the screen, the viewport displays the loaded motion capture recording as an animated 3D avatar. Below the viewport are playback controls:

  • Play/pause button — Start or stop the animation.
  • Speed selector — Choose playback speed: 0.5× (slow), 1× (normal), 1.5× (faster), 2× (double speed).

Below the controls is the timeline, a horizontal bar showing the full duration of the recording. This is where you perform trimming and task marking.

Trimming the observation window

Trimming defines the start and end points of the observation period — the portion of the recording that will be analyzed. Only lifting tasks within the trimmed range are included in the assessment.

Two vertical handle lines appear on the timeline, marking the trim range:

  • Start handle (left) — Drag to move the observation start point.
  • End handle (right) — Drag to move the observation end point.

The shaded region between the handles represents the observation window. As you adjust the handles, Motionprint recalculates the frequency based on the trimmed duration and the number of lifting tasks inside the window. Frequency is expressed as lifts per minute and must fall within NIOSH's acceptable range of 0.2–15 lifts/min.

If your frequency falls outside this range, Motionprint displays a warning and prevents the assessment from running. You can resolve this by either adjusting the trim range or reducing/increasing the number of marked tasks.

Frequency = Tasks ÷ Trimmed duration

If you have marked 5 lifting tasks in a 2-minute observation window, the frequency is 5 ÷ 2 = 2.5 lifts/min, which is within the acceptable range and will proceed normally.

Defining lifting tasks

A lifting task is a distinct lifting action that you want to assess. It is defined by a start frame and an end frame within the observation window. For each task, you must specify the load weight and the quality of the grip (coupling).

To define a task:

  1. Click and drag on the timeline to select a frame range. A blue highlight appears showing your selection.
  2. Click the Add button to create a task from that range.
  3. A task entry appears in the task list on the left. Fill in the required fields:
    • Start frame — The frame number where the lift begins (auto-populated).
    • End frame — The frame number where the lift ends (auto-populated).
    • Load weight — The weight of the object being lifted, in kilograms. Required field.
    • Coupling rating — Select Good, Fair, or Poor to describe the quality of the grip:
      • Good — The object has convenient handles or a grip surface; the lift is performed with a secure grasp.
      • Fair — The object has less-than-optimal handles or the grip is moderately awkward.
      • Poor — The object has no handles, or the grip is significantly awkward or unstable.

Once a task is created, you can edit or delete it. If you adjust the trim range to exclude a task's frame range, that task is automatically removed from the assessment.

Multiple lifting tasks and CLI

If you mark two or more lifting tasks, Motionprint switches to multi-task mode. In multi-task assessments, each task is assigned a Single-Task Lifting Index (STLI), and these are combined using the Composite Lifting Index (CLI) formula to produce a single final risk score. This is useful for assessing workstations where the operator performs multiple distinct lifting movements.

For more details on how the CLI is calculated, see the Understanding the NIOSH Assessment article.

Frequency validation

At any time, Motionprint validates your frequency against NIOSH limits. The frequency is automatically recalculated as you add, remove, or reorder tasks, and as you adjust the trim range. If the frequency is out of bounds, a warning message appears:

  • Frequency < 0.2 lifts/min — Too few tasks in the observation window. Either extend the observation window or add more tasks.
  • Frequency > 15 lifts/min — Too many tasks in the observation window. Either narrow the window or remove tasks.

The assessment cannot run until frequency is within 0.2–15 lifts/min.

Running the Assessment

Once all four steps are complete and all required fields are valid, the Run Assessment button activates and becomes clickable. Click it to submit the assessment to Motionprint's backend for processing.

A loading screen appears briefly while the data is transmitted and the NIOSH calculation is performed. Once complete, Motionprint automatically navigates to the report view, where you can review the Score Overview, Analysis, and Findings tabs.

If an error occurs during processing (invalid data, server issue, etc.), an error message is displayed and you are returned to the input page. You can then correct the issue and try again.
