Getting started

Up and running in four steps

1

Download the installer

Click the Windows button above to download the latest installer (.exe). The file is around 80 MB.

2

Run and install

Open the downloaded file and follow the setup wizard. No admin rights required for a per-user install.

3

Enter your licence key

On first launch you'll see the activation screen. Enter your licence key (format: MP-XXXX-XXXX-XXXX-XXXX). No key yet? View pricing or request a free trial.

04

Start your first assessment

Upload an MVNX file from your Xsens session and run your first ergonomic assessment. Check the documentation if you need a hand.
